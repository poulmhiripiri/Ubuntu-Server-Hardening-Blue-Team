## Example Safe Mode Decisions

- SSH port left as 22 during testing
- SSH password authentication left enabled during initial run
- Web ports not opened unless required
- IPv6 not disabled unless explicitly requested

## Follow-up Actions

2. Confirm application-specific ports and services.
3. Test SSH access from a second terminal before ending the current session.
4. Integrate logs with Wazuh, Microsoft Sentinel or Splunk.
